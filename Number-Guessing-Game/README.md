🎯 Number Guessing Game (Java Swing)

This project is a simple Number Guessing Game developed using Java Swing as part of my CodSoft Virtual Internship.

🚀 Features

- Random number between 1 and 100
- Limited number of guesses
- Hint system (Too Near, Near, Far, Too Far)
- Score tracking
- Multiple rounds support
- GUI built with Java Swing

🛠️ Technologies Used

- Java
- Java Swing
- AWT Event Handling

▶️ How to Run

1. Compile the program:
   
   javac NumberGame.java

2. Run the program:
   
   java NumberGame
