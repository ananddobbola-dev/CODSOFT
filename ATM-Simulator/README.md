🏧 ATM Simulator – Java Swing

A desktop-based ATM Simulator built using Java Swing.
This project demonstrates core banking operations like login authentication, balance enquiry, deposit, and withdrawal using file handling.

Developed as part of the CodSoft Virtual Internship.

---

✨ Key Features

- 🔐 Account number & PIN authentication
- 💰 Balance enquiry
- ➕ Deposit money
- ➖ Withdraw money
- 📁 Persistent storage using file handling
- 🖥️ Multi-screen GUI using CardLayout
- ⚡ Input validation for secure transactions

---

🧱 Project Structure

ATMSimulator/
│
├── ATM.java
├── bankAccounts.txt
└── README.md

---

▶️ How to Run

1️⃣ Compile

javac ATM.java

2️⃣ Run

java ATM

✅ Make sure bankAccounts.txt is in the same folder.


---

🛠️ Technologies Used

- Java
- Java Swing
- AWT Event Handling
- File Handling
- CardLayout

---

🔒 Validation Implemented

- Account number format check
- 4-digit PIN validation
- Numeric amount validation
- Insufficient balance protection
