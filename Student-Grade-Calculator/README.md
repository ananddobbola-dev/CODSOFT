🎓 Student Grade Calculator

📌 Description

The Student Grade Calculator is a Java console-based application developed as part of the CodSoft Virtual Internship.
It allows users to enter student details, calculate total marks, average percentage, and automatically assign grades.

🚀 Features

- Input validation for name, roll number, and marks
- Calculates total marks and average percentage
- Automatic grade assignment
- Stores multiple student records using HashMap
- Menu-driven console interface

🛠️ Technologies Used

- Java
- OOP Concepts
- Collections Framework (HashMap)

▶️ How to Run

1. Compile the program:
   javac StudentGradeCalculator.java
2. Run the program:
   java StudentGradeCalculator
